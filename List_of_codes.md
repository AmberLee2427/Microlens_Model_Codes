# List of Microlens Modeling Codes

Below is a list of publicly available gravitational microlensing modeling packages. 

* [VBMicrolensing](https://github.com/valboz/VBMicrolensing) - model generation only
* [RTModel](https://github.com/valboz/RTModel) - model fitting (VBM for generation)
* [pyLIMA](https://github.com/ebachelet/pyLIMA)
* [MulensModel](https://github.com/rpoleski/MulensModel)
* [eesunhong](https://github.com/golmschenk/eesunhong)
* [BAGLE](https://github.com/MovingUniverseLab/BAGLE_Microlensing)
* [muLAn](https://github.com/muLAn-project/muLAn)
* [microlux](https://github.com/CoastEgo/microlux)
* [microjax](https://github.com/ShotaMiyazaki94/microjax)

Comparisons of the different codes can be found in:
* [List of Model Classes Supported](https://github.com/MovingUniverseLab/Microlens_Model_Codes/blob/main/Model_Class_Support.md)
* [List of Model Fitting Approaches](https://github.com/MovingUniverseLab/Microlens_Model_Codes/blob/main/Model_Fitting_Algorithms.md)
* [Lu et al. 2026](https://scixplorer.org/abs/2025arXiv251203364L/abstract)
